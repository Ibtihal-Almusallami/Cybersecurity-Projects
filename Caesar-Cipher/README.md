# Caesar Cipher Encryptor/Decryptor

A Python-based Caesar Cipher tool that can encrypt and decrypt messages using a user-defined shift value.

## Features

- Encrypt messages
- Decrypt messages
- Supports uppercase and lowercase letters
- Preserves spaces and punctuation
- Uses modular arithmetic for alphabet wrap-around

## Technologies Used

- Python

## Example

Message: HELLO
Shift: 3

Encrypted: KHOOR
